# xyh_ml

Machine learning developments for the X &rightarrow; YH &rightarrow; bb&tau;&tau; analysis


## Quickstart

To install the dependencies, execute the following commands in your shell:

```bash
$ micromamba create -y -n xyh_ml python=3.12
$ micromamba activate xyh_ml
$ python3 -m venv venv
$ . ./venv/bin/activate
$ python3 -m pip install -r requirements.txt
```
